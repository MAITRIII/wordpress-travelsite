Companion for adding features to the Travel Agency Theme

== Description ==

Travel Agency Companion is a plugin that adds extra functionalities and features to the Travel Agency Theme by Rara Theme. The plugin when activated installs few widgets and adds extra features on the customizer. 

You need to download, install, and activate [Travel Agency](https://raratheme.com/wordpress-themes/travel-agency)Theme to use this plugin.

To get the free support, visit [here](https://raratheme.com/support-ticket/).

== Installation ==


**Method 1:**

On your WordPress admin dashboard
Visit 'Plugins > Add New'
Search for 'Travel Agency Companion' and install the plugin.
Activate 'Travel Agency Companion' from your Plugins page.

**Method 2:**

Download the plugin from WordPress.org repository
On your WordPress admin dashboard, go to 'Plugins> Add New> Upload Plugin'.
Upload the downloaded plugin file (travel-agency-companion.zip) and click 'Install Now'
Activate 'Travel Agency Companion' from your Plugins page.

= What widgets does the plugin add? =

Travel Agency Companion adds three very useful customized widgets, namely, Featured Post widget, Popular Post widget, and Recent Post widget.

= Will Travel Agency Companion work on other themes? =
 
The plugin was made to work with Travel Agency Theme made by Rara Theme. Although, the widgets could work on other themes as well.

= What features does this plugin add? =


Travel Agency Companion, besides widgets, adds extra options for customizing home page of the Travel Agency Theme.
